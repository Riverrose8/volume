## 🆕 Дополнительные полезные данные

### 1. **Метрики концентрации (Scam Detection)**

#### `dev.top_10_holder_rate` (15.24% в тесте)
- **Что это**: Процент токенов у топ-10 holders
- **Польза**: Индикатор скама (высокая концентрация = риск)
- **Использование**: 
  - Фильтр: `MAX_TOP_10_HOLDER_RATE = 50%` (если > 50% = скам)
  - Отображение в алерте: "Top 10 Holders: 15.24%"
- **Статус**: ⭐ **ВЫСОКИЙ ПРИОРИТЕТ**

#### `dev.creator_token_balance` (0 в тесте)
- **Что это**: Баланс токенов у создателя
- **Польза**: Если создатель держит много токенов = риск rug pull
- **Использование**: Фильтр или предупреждение в алерте
- **Статус**: ⭐ **СРЕДНИЙ ПРИОРИТЕТ**

### 2. **Метрики ликвидности**

#### `liquidity` (10.98 BNB в тесте)
- **Что это**: Текущая ликвидность в BNB
- **Польза**: Дополнительная проверка ликвидности (может отличаться от DexTools)
- **Использование**: 
  - Валидация данных DexTools
  - Отображение в алерте как альтернативный источник
- **Статус**: ⭐ **СРЕДНИЙ ПРИОРИТЕТ**

#### `pool.liquidity` (10.98 BNB в тесте)
- **Что это**: Ликвидность пула
- **Польза**: Детальная информация о пуле
- **Использование**: Анализ пула ликвидности
- **Статус**: ⚠️ **НИЗКИЙ ПРИОРИТЕТ** (дублирует основное поле)

#### `pool.fee_ratio` ("0" в тесте)
- **Что это**: Комиссия пула (обычно 0.25% или 0.3% для PancakeSwap)
- **Польза**: Проверка типа пула
- **Использование**: Валидация пула
- **Статус**: ⚠️ **НИЗКИЙ ПРИОРИТЕТ**

## 💡 Примеры использования

### Пример 1: Улучшенная фильтрация скама

```python
# Новые фильтры на основе Apify данных
MAX_TOP_10_HOLDER_RATE = 50  # Если > 50% = скам
MAX_CREATOR_OPEN_COUNT = 10  # Если создатель создал > 10 токенов = скаммер
MAX_IMAGE_DUP_COUNT = 0  # Если есть дубликаты изображения = скам

if gmgn_data.get('top_10_holder_rate', 0) > MAX_TOP_10_HOLDER_RATE:
    logger.info(f"🚫 High top 10 holder rate: {gmgn_data['top_10_holder_rate']}%")
    continue

if gmgn_data.get('creator_open_count', 0) > MAX_CREATOR_OPEN_COUNT:
    logger.info(f"🚫 Creator is a scammer: {gmgn_data['creator_open_count']} tokens created")
    continue
```

### Пример 2: Улучшенный алерт

```python
# Добавить в format_alert():
if token.get('top_10_holder_rate'):
    text += f"<b>📊 Top 10 Holders:</b> {token['top_10_holder_rate']:.1f}%\\n"

if token.get('buy_sell_ratio'):
    text += f"<b>📈 Buy/Sell Ratio:</b> {token['buy_sell_ratio']:.2f}\\n"

if token.get('creator_open_count', 0) > 5:
    text += f"⚠️ <b>Creator created {token['creator_open_count']} tokens</b>\\n"
```

### Пример 3: Валидация данных

```python
# Сравниваем данные из разных источников
dextools_holders = token_data.get('holders_count', 0)
gmgn_holders = gmgn_data.get('holder_count', 0)

if abs(dextools_holders - gmgn_holders) > dextools_holders * 0.2:  # Разница > 20%
    logger.warning(f"⚠️ Holders mismatch: DexTools={dextools_holders}, GMGN={gmgn_holders}")
```
