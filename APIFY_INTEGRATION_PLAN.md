# План интеграции Apify для получения GMGN данных

## Текущая ситуация

1. ✅ Функция `fetch_gmgn_token_data()` уже создана с парсингом HTML
2. ⚠️ Парсинг HTML может не работать из-за защиты GMGN
3. ❓ Неизвестно, есть ли bundler % и total fees в Apify акторах для BSC

## План действий

### Шаг 1: Тестирование Apify актора (ПЕРВЫЙ ПРИОРИТЕТ)

**Цель**: Проверить, какие данные возвращает Apify актор для BSC токена

**Действия**:
1. Получить Apify API токен:
   - Зарегистрироваться на https://apify.com
   - Перейти в https://console.apify.com/account/integrations
   - Создать API token
   - Добавить в `.env`: `APIFY_API_TOKEN=your_token_here`

2. Запустить тестовый скрипт:
   ```bash
   python test_apify_gmgn.py
   ```

3. Проанализировать результаты:
   - Проверить файл `apify_test_result_*.json`
   - Найти поля, связанные с bundler и total fees
   - Определить, есть ли эти данные для BSC

### Шаг 2: Интеграция Apify API (если данные есть)

**Если тест показал наличие bundler/total fees**:

1. ✅ Функция `fetch_gmgn_via_apify()` уже создана
2. ✅ Функция `fetch_gmgn_token_data()` уже обновлена для использования Apify как приоритет
3. Нужно только добавить `APIFY_API_TOKEN` в `.env`

**Преимущества**:
- Более надежно, чем парсинг HTML
- Не зависит от изменений структуры страницы
- Меньше вероятность блокировки

### Шаг 3: Альтернативные решения (если данных нет)

**Если bundler/total fees НЕТ в Apify данных**:

#### Вариант A: Использовать альтернативные индикаторы
- `dev.top_10_holder_rate` - высокая концентрация может указывать на скам
- `maker_token_tags` с тегом "bundler" - индикатор наличия bundler
- Комбинация других метрик для определения скама

#### Вариант B: Комбинированный подход
- Использовать Apify для получения comprehensive данных
- Парсить HTML только для bundler % и total fees (если критично)
- Кешировать результаты Apify на 1-2 часа

#### Вариант C: Улучшить существующие фильтры
- Использовать данные из Apify для улучшения других проверок
- Не полагаться только на bundler/total fees
- Комбинировать несколько индикаторов

## Стоимость использования

### При использовании только для токенов, прошедших фильтры:
- **Низкая активность**: ~$32-119/месяц
- **Средняя активность**: ~$65-238/месяц  
- **Высокая активность**: ~$130-475/месяц

### Оптимизация:
- **Кеширование** (1-2 часа): экономия 50-70%
- **Только подозрительные токены**: экономия 70-80%
- **Использовать только Token Stat Scraper**: экономия ~50%
